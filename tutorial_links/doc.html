Data Initializing

When creating your maps you can enter a large subset of coordinates to visually plot data. Also make sure your final CSV size is no larger than 1GB. This is for only the free version of Mapbox.

Make sure that the csv is in first normal form; has all entires are indivisible. Example below shows the difference between a CSV in first normal and not in first normal form.

Don't! (Not in 1NF)

Latitude Longitude Time
40.7687121 -73.966227 12:03:00,12:10:01,02:38:12
40.75896 -73.985195 19:09:88
40.766372 -73.957579 23:00:09,15:04:23

Do! (In 1NF)

Latitude Longitude Time
40.7687121 -73.966227 12:03:00
40.7687121 -73.966227 12:10:01
40.7687121 -73.966227 02:38:12
40.75896 -73.985195 19:09:88
40.766372 -73.957579 23:00:09
40.766372 -73.957579 15:04:23

(!) Also another thing to note is that make sure when you have your columns, name your time column 'time' otherwise mapbR's filter input bar will not filter for time values correctly!

(!) Also please label your columns on the top of the CSV file as well! (i.e. time,lat,lon)

(!) Format for 'time' column is xx:xx:xx.

mapbR

Although Mapbox has its own section to visualize all coordinates on a map and filter for points, it still lacks a bit of interactivity with its filter and shareablity. Here's where mapbR comes in.

This section below will cover how to use mapbR after all the data is setup in Mapbox. You must store your data in MapBox in order to pull map data using mapbR!

Properties (Form)

So mapbR has a couple of different forms you need to fill in before you get to your html map. Here's where to get all of them. To see it in action head over to the Example tab.

MapBox Settings

These are the properties to find your Mapbox maps off the Mapbox APIs.

  • Access Token: Go into Account > Access Token > Create New Token. Must have all 4 Public scopes checked. Copy and paste the string given.
  • Map URL: This can be any name.
  • Map ID: Go into Tilesets and then click into the Tileset you are using. Then on the right hand will be Map ID. Copy and paste the string below it. You MUST have "mapbox://" before the MapID!
  • Source Layer: Go into Tilesets and then click into the Tileset you are using. Copy and paste the name. Likewise you can find it by going into the Tileset and right under Layer Details.
  • Property: Go into Tilesets and then click into the Tileset you are using. Then under Layer details, choose the property (aka column too) name that you chose to count up your occurences. If you don't have any (i.e. because each plot can be a single occurence), put anything in there to bypass the blank error.

    • Slider Filter Settings

    This part could be a bit confusing, but this is only for the slider on mapbR to filter out what happens at the time of day using the slider.

    For example, if you want only points to show every 2 hours, then you would put:

    * 120 into Amount Split

    * Minutes into Time Unit

    If you want every 1 minute 30 seconds per tick on the filter, then you would do 90 seconds.

    Cool tip: you can hold the left or right arrow key on the slider to make it look like it's animating throughout the day.

    Map View Settings

    This is just simply where you want the map to be centered when you create the map. If the zoom level is increasing, it means it's getting closer and closer into the map.
